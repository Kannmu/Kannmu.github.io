Acoustic Haptics Array (UMH v5.0)

Architecting an ultrasonic phased array utilizing a hexagonal close-packed layout (60 elements). Designed to maximize Rayleigh wave strain on the skin, delivering high-fidelity, mid-air tactile perception for immersive VR environments.

BCI Secure Comms (BSTCM)

Pioneered a paradigm integrating Brain-Computer Interfaces with Space-Time-Coding Metasurfaces. Fusing visual stimulus encoding with physical-layer security to create highly reliable, encrypted wireless neural communication channels.

Pressure-Sensing Blink HCI

Developed an alternative to infrared eye-tracking by utilizing pressure sensors for voluntary blink interactions. This method mitigates hardware cost and ocular hazards while establishing new ergonomic heuristics for interaction designers.
