Stimulation Modes

Multiple Haptic Patterns

UMH supports various stimulation modes for different haptic experiences.

AM

Amplitude Modulation

200Hz amplitude modulation creates continuous tactile sensation at focal point.

LM

Linear Motion

Focal point moves along linear trajectory, creating sliding haptic sensation.

STM

Spatio-Temporal Modulation

High-frequency spatial modulation creates sharp, localized tactile points.

TwinTrap

Dual Focus

Two counter-propagating focal points create acoustic trapping for particle manipulation.

Circular DMA Mode

Direct Memory Access in circular mode enables continuous waveform generation with zero CPU overhead. The DMA controller automatically transfers phase data to timer registers at 40kHz.

Applications

Transforming Industries

From automotive to healthcare, mid-air haptics is redefining human-machine interaction.

Automotive

Gesture control without contact. Drivers can interact with infotainment systems through mid-air haptic feedback, reducing visual distraction.

VR/AR

Tactile feedback in virtual environments. Feel virtual objects, textures, and interactions without wearing gloves or holding controllers.

Medical

Sterile interface interaction. Surgeons and medical staff can control equipment without touching surfaces, maintaining sterile fields.

Industrial

HMI for cleanroom environments. Operators can interact with control systems without introducing contaminants or wearing special equipment.
